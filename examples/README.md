## Available Examples

### `convex-test.tsx`
A complete example showing:
- How to use Convex queries (`useQuery`)

- Real-time data updates
- Form handling with Convex

### `database-viewer.tsx`
A database viewer component showing:
- How to display data from Convex
- Database statistics
- Table view of documents
- Raw JSON data display

## How to Use

1. **Copy the example** to your `app/components/` or `components/` folder
2. **Import and use** in your pages:
   ```tsx
   import { ConvexTest } from "@/examples/convex-test";
   ```
3. **Customize** for your needs
